int 2000

是否开启动画的阈值，当单个系列显示的图形数量大于这个阈值时会关闭动画。

AnimationStyle.type​

AnimationType

动画类型。

可选：

  • Default: 默认。内部会根据实际情况选择一种动画播放方式。
  • LeftToRight: 从左往右播放动画。
  • BottomToTop: 从下往上播放动画。
  • InsideOut: 由内到外播放动画。
  • AlongPath: 沿着路径播放动画。当折线图从左到右无序或有折返时，可以使用该模式。
  • Clockwise: 顺时针播放动画。
